Este site foi desenvolvido com o objetivo de organizar, analisar e visualizar dados públicos de dengue no Brasil de forma clara e comparável entre estados. Os dados utilizados são provenientes do portal oficial de dados abertos do governo federal, disponibilizados pelo Ministério da Saúde, dentro da plataforma Dados Abertos do SUS. As informações são extraídas do Sistema de Informação de Agravos de Notificação (Sinan), sistema responsável pelo registro nacional de doenças de notificação compulsória, como a dengue.
